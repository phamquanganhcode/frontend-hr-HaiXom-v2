1. Clone dự án: https://github.com/NguyenTruongNT/frontend-hr-HaiXom.git

2. Cài đặt thư viện: 
- npm install (Nếu dùng npm)
  
  Hoặc yarn install (Nếu dùng yarn)
- npm install lucide-react
- npm install axios react-router-dom @reduxjs/toolkit react-redux
3. Cài đặt UI Library: 
- npm install -D tailwindcss postcss autoprefixer
- npx tailwindcss init -p
4. Thư viện hỗ trợ hiển thị (Biểu đồ, Icon)
- npm install chart.js react-chartjs-2 lucide-react
5. Chạy dự án: 
- npm start hoặc npm run dev
